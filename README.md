# CSE 3241 Project

## Dependencies

- Install Python 3
- Install [Faker](https://github.com/joke2k/faker)
```
pip3 install Faker
```

## Usage

To execute the sample queries:
```
python3 main.py
```

The [a.db](a.db) file is already pre-populated with data.
If you want to re-populate the database then run:
```
python3 init.py
```
